Overview

Represents a context to the message broker.

You should only need to create a Context directly if you are connecting to a remote broker, you need transactional semantics, or you need better perfomance when doing lots of Destination#publish or Destination#receive calls in rapid succession, since each call will create and close a context if one is not provided.

- (Context) initialize(options = {}, &block)

Creates a new context.

You are responsible for closing any contexts you create.

If given a block, the Context instance will be passed to the block and the Context will be closed once the block returns.

Parameters:

  • options (Hash) (defaults to: {})

Options Hash (options):

  • :mode (Symbol) — default: :auto_ack —

    One of: :auto_ack, :client_ack, :transacted

  • :client_id (String) —

    Identifies the client id for use with a durable topic subscriber.

  • :host (String) —

    The host of a remote broker.

  • :port (Number) — default: nil, 5445 if :host provided —

    The port of a remote broker.

  • :username (String) —

    The username for the remote broker.

  • :password (String) —

    The password for the remote broker.

  • :remote_type (Symbol) — default: :hornetq_standalone —

    When connecting to a HornetQ instance running inside WildFly, this needs to be set to :hornetq_wildfly.

  • :reconnect_attempts (Number) — default: 0 —

    Total number of reconnect attempts to make before giving up (-1 for unlimited).

  • :reconnect_retry_interval (Number) — default: 2000 —

    The period in milliseconds between subsequent recontext attempts.

  • :reconnect_max_retry_interval (Number) — default: 2000 —

    The max retry interval that will be used.

  • :reconnect_retry_interval_multiplier (Number) — default: 1.0 —

    A multiplier to apply to the time since the last retry to compute the time to the next retry.

def initialize(options = {}, &block)
  validate_options(options, CONTEXT_OPTIONS)
  options[:mode] = coerce_mode(options[:mode])
  create_options = extract_options(options, WBMessaging::CreateContextOption)
  @internal_context = default_broker.create_context(create_options)
  if block
    begin
      block.call(self)
    ensure
      close
    end
  end
end

def mode
  case @internal_context.mode
  when WBContext::Mode::AUTO_ACK
    :auto_ack
  when WBContext::Mode::CLIENT_ACK
    :client_ack
  when WBContext::Mode::TRANSACTED
    :transacted
  end
end
