+ (Queue) queue(name, options = {})

Creates a new queue reference.

This may be a reference to a remote or local (in-vm) queue. Obtaining a reference to an in-vm queue will cause the queue to be created within the broker if it does not already exist. For remote queues, the queue must already exist in the remote broker.

If a context is provided, it must be remote, and will be remembered and used by any method that takes a :context option.

Parameters:

  • name (String) —

    The name of the queue.

  • options (Hash) (defaults to: {}) —

    Options for queue creation.

Options Hash (options):

  • :context (Context) —

    A context to a remote broker to use; caller expected to close.

  • :durable (true, false) — default: true —

    Whether messages persist across restarts.

  • :selector (String) —

    A JMS (SQL 92) expression to filter published messages.

  • :default_options (Hash) —

    A set of default options to apply to any operations on this queue.

Returns:

  • (Queue) —

    The queue reference.
